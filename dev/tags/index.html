ColorProfiles.Tags.AToB0Tag — Type
AToB0Tag{T}

AToB0Tag is the alias for AToBTag{Signature(:A2B0)}.

This tag describes a color transform depending on the device/profile class. Except for AbstractClass and DeviceLinkClass, the transform is mainly for Parceptual rendering.

ColorProfiles.Tags.MediaWhitePointTag — Type
MediaWhitePointTag = XYZTag{sig"wtpt"}

MediaWhitePointTag is the alias for XYZTag{Signature(:wtpt)}.

This tag specifies the chromatically adapted CIEXYZ values of the media white point. The xyz field is in normalized CIEXYZ format (i.e. xyz.y == 1.0).
